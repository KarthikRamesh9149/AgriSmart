================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: Modal doesn't appear after clicking Generate
  → Check browser console (F12 → Console)
  → Look for red errors
  → Verify CSV was actually uploaded

PROBLEM: Feature 2 score isn't colour-coded
  → Hard refresh: Ctrl+Shift+R
  → Score should be green/yellow/red in header
  → Check "Min. for sustainability: 60 pts" row

PROBLEM: Modal appears but no content shows
  → Modal might not be scrollable
  → Try scrolling down with mouse wheel
  → Or check browser console for errors

PROBLEM: Graph bars not showing
  → Hard refresh browser: Ctrl+Shift+R
  → Check if you can scroll down in modal
  → CSS might not have loaded

PROBLEM: PDF export fails
  → Library html2pdf.js should already be available
  → Check console for error messages
  → PDF export is optional - main features are modal + graph
