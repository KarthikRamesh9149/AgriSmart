BUG #2: CSV GENERATE RETURNS 500 ERROR
════════════════════════════════════════════════════════════════════════════════

BEFORE (❌ BROKEN):
─────────────────────────────────────────────────────────────────────────────────
User uploads CSV:  ✅ File accepted, parsed
User clicks Generate: ⏳ Loading...
Server receives request: ✅ Request valid
Mistral API called: ❌ API fails or times out
Server responds: 💥 CRASH - No try-catch!

Result: Browser shows "Error: API error: 500 Internal Server Error"
        Button stuck in loading state
        User has no idea what went wrong


AFTER (✅ FIXED):
─────────────────────────────────────────────────────────────────────────────────
User uploads CSV:  ✅ File accepted, parsed
User clicks Generate: ⏳ Loading...
Server receives request: ✅ Request valid
Mistral API called: ⚠️  API fails or times out
Server catches error: ✅ Try-catch block catches it
Graceful fallback: ✅ Returns analysis JSON anyway

Result: Browser shows "Policy analysis generated in fallback mode"
        Shows row count, columns, district name
        Always returns 200 OK (no 500 error)


CODE CHANGE (CSV Error Handling):
─────────────────────────────────────────────────────────────────────────────────
File: server/src/interfaces/http/routes/llm.ts

BEFORE:
  fastify.post('/llm/policy-freeform', async (request, reply) => {
    const data = parseRequest(request);

    if (container.aiService.isAvailable(3)) {
      const analysis = await container.aiService.analyzePolicyData({...});
      // ❌ No try-catch! If API fails, entire function crashes
      return reply.send({ analysis, ... });
    }

    return reply.send({ analysis: fallback });
  });

AFTER:
  fastify.post('/llm/policy-freeform', async (request, reply) => {
    try {                                          // ✅ Wrap in try-catch
      const data = parseRequest(request);

      if (container.aiService.isAvailable(3)) {
        try {                                      // ✅ Inner try for API
          const analysis = await container.aiService.analyzePolicyData({...});
          return reply.send({ analysis, ... });
        } catch (aiError) {
          logger.error({...}, 'Policy analysis failed');
          // ✅ Fall through to fallback
        }
      }

      // ✅ Always reached (fallback)
      return reply.send({ analysis: fallback });
    } catch (error) {
      logger.error({...}, 'Endpoint error');
      return reply.status(500).send({
        error: { code: 'INTERNAL_SERVER_ERROR', message: '...' }
      });
    }
  });

Impact: No more uncaught exceptions, always graceful fallback
